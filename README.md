# Graduation_thesis_main
## Steps to run the project
- Clone source code by HTTP or SSH key in Github
- Point to the be (Backend) folder 
- Open new terminal 
```shell
npm install
```
or
```shell
yarn
```
- Point to the fe (Frontend) folder 
- Open new terminal 
```shell
npm install
```
or
```shell
yarn
```
- when done open new terminal and run command
```shell
npm start
```
